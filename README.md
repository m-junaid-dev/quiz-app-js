# Quiz App (JavaScript)

This is a simple quiz application built using HTML, CSS, and JavaScript. It allows users to answer multiple-choice questions, calculates the score, and displays the result with a smooth animated percentage counter.

## Features

- Multiple choice quiz
- Score calculation
- Animated percentage result display
- Form validation (ensures all questions are answered)
- Smooth UI interaction

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

1. User selects answers for all questions.
2. On form submission:
   - The app checks if all questions are answered.
   - It calculates the total correct answers.
   - Converts the score into percentage.
3. The result is displayed with an animated counter.

## Project Structure

- index.html – main structure of the quiz
- style.css – styling (optional if separated)
- script (inside HTML) – quiz logic and animation

## How to Run

1. Download or clone the repository.
2. Open the `index.html` file in your browser.
3. Start answering the quiz.
